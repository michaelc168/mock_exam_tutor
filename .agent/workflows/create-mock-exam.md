## 步驟

### 1. 需求確認
- 若使用者未指定：
  - 預設年級：小六升國一。
  - 科目：國語、英數、數學（各 10 題）。
- 根據使用者輸入，決定科目與題目數、預期難度。

### 2. 生成題目
- 依照 `exams/templates/exam-template.md` 的結構：
  - 先填寫試卷基本資訊（年級、科目、時間、滿分）。
  - 逐一產生題目，確保考點標註完整。
- 題目不是只拿 `exams/bank` 含子目錄下的一比一的內容，而是要產出類似的題目。
- `exams/bank` 含子目錄下的所有內容都是可以參考的對象，而不是只專注於單一檔案而已，產出的題目要多樣化，而不是拿現有的題目進行排列組合而已。
- 確保每題都附上標準答案。
- 數學會有方程式的問題，例如：$3 \times (y - 5) + 8 = 20$，所以在產 .md 時可以維持，因為 .md 是文字格式；但在產 PDF 時，則要把前述的方程式利用像 auto latex equations 的extension 把這類的方程式轉成小六生看的懂的內容，也方便列印 (printable)。

### 3. 產出解析
- 對每一題撰寫簡要解析：
  - 1～3 句說明考點與解法關鍵。
- 將題目、答案、解析匯總成一份 Markdown 檔案，存放於 `exams/generated/`。

### 4. 校驗與輸出
- 自行檢查題號、答案選項是否對齊。
- 將試卷主體內容作為回覆輸出，並註明檔案預計存放路徑。
