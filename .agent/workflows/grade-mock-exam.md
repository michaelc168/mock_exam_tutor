## 步驟

### 1. 載入資料
- 讀取指定的試卷檔案（題目與標準答案）。
- 讀取指定的學生作答檔案（Q1: ..., Q2: ...）。

### 2. 對題批改
- 逐題比對學生答案與標準答案。
- 對每題產出：
  - 對 / 錯標記。
  - 若錯誤，簡述錯誤類型（例：計算錯誤、看錯條件、概念誤解）。

### 3. 彙總成績
- 計算：
  - 總分。
  - 各科分數與正確率。
- 產出錯題清單，包含題號、考點、錯誤原因摘要。

### 4. 回饋輸出
- 以 Markdown 表格呈現成績與錯題清單。
- 給出 3～5 條具體的學習建議（優先針對出錯最多的考點）。
